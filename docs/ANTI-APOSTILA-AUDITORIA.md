# ANTI APOSTILA AUDITORIA
Data: 2026-03-07
Status: obrigatorio em revisoes de UX

Objetivo: eliminar qualquer sinal de "ler, clicar e avancar" como espinha do jogo.

## Itens identificados
1. Intro com dependencia de texto contextual.
Acao: reduzir.
Destino: transformar contexto em evento visual reativo.

2. Toolkit como lista de selecao obrigatoria.
Acao: transformar.
Destino: introduzir escolha com custo e sinergia.

3. Avatar com fluxo de confirmacao tradicional.
Acao: transformar.
Destino: decisao expressiva com resposta dramatica imediata.

4. Result com bloco de resumo informativo.
Acao: reduzir.
Destino: enfatizar payoff emocional e convite de replay.

5. Termos muito institucionais expostos sem camada diegetica.
Acao: transformar.
Destino: apresentar os mesmos conceitos como poderes/artefatos narrativos.

## Checklist anti apostila para novas telas
1. Existe acao jogavel em menos de 5 segundos?
2. Existe feedback visual e sonoro imediato?
3. O jogador pode errar e se recuperar?
4. A tela pode ser entendida sem ler um paragrafo?
5. Existe movimento de mundo, nao apenas de UI?

Se 2 ou mais respostas forem "nao", a tela precisa refatorar.
